# 📈 Stock Portfolio Tracker Pro

## 📌 Description
Stock Portfolio Tracker Pro is a Python application that helps users manage stock investments. It allows users to add stocks, calculate investment values, analyze portfolios, and export reports.

## ✨ Features
- Investor Details
- Currency Selection
- Stock Search
- Add, Remove & Update Stocks
- Portfolio Summary
- Total Investment Calculation
- Highest & Lowest Investment
- Top 3 Investments
- Profit/Loss Calculator
- Save Report as TXT
- Save Report as CSV

## 🛠️ Technologies Used
- Python
- Dictionaries
- CSV Module
- File Handling
- Functions
- Loops
