# 🎮 Hangman Ultimate

## Description
Hangman Ultimate is a Python console game developed as part of the Horizon TechX Python Internship.

## Features
- Multiple Categories
- Difficulty Levels
- Hint System
- Reveal Letter
- ASCII Hangman
- Score System
- Leaderboard
- Achievements
- High Score Saving
- Play Again Option

## Technologies Used
- Python
- Random Module
- File Handling
- Loops
- Functions
