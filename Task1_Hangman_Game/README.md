# 🎮 Hangman Ultimate

## 📌 Description
Hangman Ultimate is a Python console-based word guessing game developed as part of the Horizon TechX Python Internship. Players guess letters to reveal a hidden word while managing limited lives.

## ✨ Features
- Multiple Categories
- Difficulty Levels
- Hint System
- Reveal Letter Feature
- ASCII Hangman Graphics
- Score System
- Game Statistics
- Leaderboard
- High Score Saving
- Play Again Option

## 🛠️ Technologies Used
- Python
- Random Module
- File Handling
- Loops
- Functions
- Conditional Statements
