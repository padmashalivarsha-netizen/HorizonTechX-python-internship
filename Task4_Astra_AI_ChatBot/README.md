# 🤖 Astra AI ChatBot

## 📌 Description
Astra AI ChatBot is an interactive Python-based chatbot that responds to user queries, shares jokes, motivational quotes, fun facts, and maintains chat history.

## ✨ Features
- Interactive Conversations
- Personalized Greetings
- AI Knowledge Base
- Random Jokes
- Motivational Quotes
- Fun Facts
- Date & Time
- Help Menu
- Chat History
- Session Summary

## 🛠️ Technologies Used
- Python
- Random Module
- Datetime Module
- File Handling
- Functions
- Dictionaries
